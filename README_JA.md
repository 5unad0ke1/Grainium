# インストール

## Package Manager経由
1. Unity Package Managerを開く
2. [`+`]ボタン → `add package from git URL...`
3. 以下のURLを入力：

```
https://github.com/5unad0ke1/Grainium.git?path=/Grainium/Assets/Grainium
```
